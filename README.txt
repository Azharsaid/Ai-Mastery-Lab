AI Mastery Lab (Offline Workshop Website)

How to run:
1) Keep index.html, style.css, script.js in the same folder.
2) Put your media here:
   assets/Images and videos/
   - Images: 1.png ... 28.png
   - Videos: 1.mp4 ... 10.mp4
3) Open index.html in Chrome/Edge.

Notes:
- The site works offline using localStorage (progress, saved prompts, scoreboard).
- The Vault will auto-hide missing media files.
